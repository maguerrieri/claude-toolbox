Run the diverse-ideation loop for: **$ARGUMENTS**

Use the `generate` skill. Parse "$ARGUMENTS" into `<type>` (the content type,
mapping to its reservoir at `docs/generation/<type>.md`), an optional **count**
(default ~15–30), and an optional **mode** (`generate` default · `cluster` ·
`frame`).

Do the skill's **Step 0 first** — select the Promotion adapter (project memory →
repo `CLAUDE.md` `Promotion:` line → default `inline`) — then run the loop: read
the type's morphological frame → 3–4 independent, lens-varied passes (parallel,
blind to each other) → axis-tag → dedupe/append (never overwrite; judgment off).
If the type has no frame yet, run `frame` first. Don't decide or rank — a
contested item promotes via the adapter.
