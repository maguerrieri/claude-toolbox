# gm Forge + Promote-Seal (PR 2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the GM use the `generate` plugin as a deep oracle — forge a diverse pool into a rollable campaign table, and converge-and-seal one canonical secret behind the GM screen — building on the merged table-system foundation.

**Architecture:** The forge is mostly **skill instructions + declarative data**, with a thin deterministic core. The GM (model-side) invokes the `generate` skill, which writes a reservoir; a new `bin/forge harvest` CLI then deterministically turns that reservoir into a foundation table (`tables/<type>.md`, or sealed `.gm/tables/<type>.md`). Frames are **genre data, like oracles** — they live in each `adapters/<name>/frames/` (not the core), validated by `validate-adapter --all`; the forge resolves the active adapter's frame, falls back to generic's neutral baseline, else frames on the fly. A **`campaign` Promotion adapter** routes generate's converge-seam into `campaign gm-seal` (the GM screen). `/gm:forge` + SKILL prose wire it together with prep/on-demand modes and graceful degradation when `generate` is absent.

**Tech Stack:** Python 3 stdlib only (`argparse`, `re`, `os`); pytest via `uvx pytest`; the `generate` plugin (soft dep); the merged foundation (`bin/roll table`, `bin/campaign gm-seal`/`gm-reveal`, `tables/` + `.gm/tables/`).

## Global Constraints

- Python **standard library only** in `bin/`.
- Commit message format: `[#11] (Claude Code + Opus 4.8) <scope>: <description>`.
- **Soft coupling:** nothing in `bin/` imports or hard-requires `generate`; the forge degrades to a GM-improvised list when it's absent. The foundation never depends on `generate`.
- A **sealed** table/secret is written via a Bash CLI (`bin/forge`, `bin/campaign`), never the Write tool, so it collapses behind the GM screen.
- Bump `plugins/gm/.claude-plugin/plugin.json` `version` **0.3.0 → 0.4.0** in this PR (version-gated installs — repo CLAUDE.md → Releasing).
- Tests run from the plugin dir: `cd plugins/gm && uvx pytest tests -q`. Foundation pieces (`roll table`, `gm-seal`/`gm-reveal`) already exist and are tested — reuse, don't re-test.
- The irreducibly-LLM parts (invoking `generate`, diversity quality, `/gm:forge` prose) are authored + reviewed, and validated by a **live forge run** at FINISH (smoke), not a committed test.

```python
#!/usr/bin/env python3
"""gm forge — harvest a generate reservoir into a rollable table.

  forge harvest <reservoir-file> <table-file>

Reads the markdown under the reservoir's `## Reservoir` heading (the diverse pool
`generate` produced) and writes it to <table-file> as a foundation table — one
`- ` entry per candidate, block-rule continuations preserved. The reservoir is
scratch; the table is the durable, rollable artifact. Run via Bash (not the Write
tool) so a sealed table (`.gm/tables/...`) stays collapsed behind the GM screen.
"""
import argparse
import os
import re
import sys

HEADING = re.compile(r"^##\s+", )
RESERVOIR = re.compile(r"^##\s+Reservoir\b", re.I)


def reservoir_block(text):
    """Lines under `## Reservoir`, up to the next `## ` heading, blank-trimmed."""
    out, in_res = [], False
    for line in text.splitlines():
        if RESERVOIR.match(line):
            in_res = True
            continue
        if in_res and HEADING.match(line):
            break
        if in_res:
            out.append(line.rstrip())
    while out and not out[0].strip():
        out.pop(0)
    while out and not out[-1].strip():
        out.pop()
    return out


def cmd_harvest(a):
    with open(a.reservoir) as f:
        block = reservoir_block(f.read())
    if not any(line.startswith("- ") for line in block):
        sys.exit(f"forge: no '## Reservoir' entries in {a.reservoir}")
    parent = os.path.dirname(os.path.abspath(a.table))
    os.makedirs(parent, exist_ok=True)
    title = os.path.splitext(os.path.basename(a.table))[0]
    with open(a.table, "w") as f:
        f.write(f"# {title}\n" + "\n".join(block) + "\n")
    n = sum(1 for line in block if line.startswith("- "))
    print(f"forge: harvested {n} entries -> {a.table}")


def main(argv=None):
    ap = argparse.ArgumentParser(prog="forge")
    sub = ap.add_subparsers(dest="cmd", required=True)
    p = sub.add_parser("harvest"); p.add_argument("reservoir"); p.add_argument("table")
    a = ap.parse_args(argv)
    {"harvest": cmd_harvest}[a.cmd](a)


if __name__ == "__main__":
    main()
```

```python
AXIS_ROW = re.compile(r"^\|\s*([^|]+?)\s*\|\s*([^|]+?)\s*\|\s*$")


def validate_frame(path):
    name = os.path.basename(path)
    text = open(path).read()
    errs = []
    m = re.search(r"^##\s+Axes\b(.*?)(^##\s+|\Z)", text, re.S | re.M | re.I)
    if not m:
        return [f"frame {name}: missing '## Axes' section"]
    axes = []
    for line in m.group(1).splitlines():
        r = AXIS_ROW.match(line)
        if not r:
            continue
        axis, values = r.group(1).strip(), r.group(2).strip()
        if axis.lower() in ("axis", "") or set(values) <= set("-| "):
            continue                                   # header / separator row
        vals = [v.strip() for v in values.split(",") if v.strip()]
        axes.append((axis, vals))
    if not 4 <= len(axes) <= 6:
        errs.append(f"frame {name}: has {len(axes)} axes (need 4-6)")
    for axis, vals in axes:
        if not 3 <= len(vals) <= 5:
            errs.append(f"frame {name}: axis '{axis}' has {len(vals)} values (need 3-5)")
    shape = re.search(r"^##\s+Shape\b(.*?)(^##\s+|\Z)", text, re.S | re.M | re.I)
    if not shape or not shape.group(1).strip():
        errs.append(f"frame {name}: missing or empty '## Shape' section")
    return errs
```

**generic — neutral baseline (6):**

- **oddity** — form (growth, creature, structure, weather, mark) · wrongness (too still, too many, it sings, it remembers, it grows wrong) · origin (a disaster, an old place, unknown, a vow) · use (heals, harms, warns, lures). Shape: one weird thing this world produced.
